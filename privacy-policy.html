Ḥanafī Atlas — Privacy Policy

Last updated: 25 June 2026

Overview

Ḥanafī Atlas is a free, open-source educational web application that traces 1,400 years of Ḥanafī scholarship through an interactive timeline, map, and scholar network. The app is a Progressive Web App (PWA) that works entirely in your browser — it does not collect, store, or transmit any personal data.

Data We Collect

None. Ḥanafī Atlas does not:

  • Collect personal information (names, emails, phone numbers)
  • Require account registration or login
  • Use cookies for tracking or analytics
  • Send data to any server
  • Use advertising SDKs or third-party trackers

Local Storage

The app uses your browser's built-in service worker and cache storage to save the app's files (HTML, CSS, icons) locally so it works offline. This data never leaves your device. You can clear it at any time via your browser settings.

Third-Party Resources

The app loads the following third-party resources when online:

  • Map tiles — OpenFreeMap (OpenStreetMap-based) for the interactive map view. These are loaded directly by your browser from tiles.openfreemap.org.
  • Fonts — Google Fonts (Amiri, Inter) for typography.
  • JavaScript libraries — MapLibre GL JS and D3.js from unpkg.com.

These requests are made directly by your browser to these CDN endpoints. No personal data is sent with these requests beyond standard HTTP headers (IP address, user agent), which are governed by those providers' own privacy policies.

Offline Mode

Once loaded, the app caches its core files and can be used fully offline. The map view requires an internet connection only for loading new map tiles. The timeline and network graph work completely offline.

Children's Privacy

Ḥanafī Atlas is an educational resource suitable for all ages. It does not knowingly collect any data from anyone, including children. No parental consent is required because no data is collected.

Open Source

The entire source code is publicly available at github.com/ziyadasghar-GIT/hanafi-atlas under the CC BY 4.0 license. You can audit the code yourself to verify this privacy policy.

Changes to This Policy

If we ever change this privacy policy, we will update this page and the "Last updated" date above.

Contact
